Tres elementos b√°sicos de cualquier gr√°fico

1) data: datos -> tips

2) aes: un conjunto de mapeos est√©ticos entre las variables en los datos y las propiedades visuales -> totbill mapeado en el eje de las x , tip mapeado en el eje de las y

3) layer: al menos una capa que describe como representar cada observaci√≥n, las capas son creadas con la funci√≥n geom -> puntos mediante geom_point

Faceteado

  • Se pueden desplegar variables categ√≥ricas adicionales en un plot usando faceteado.

  • Crea tablas de gr√°ficos partiendo los datos en subconjuntos y mostr√°ndolos el mismo gr√°fico para cada subconjunto.

Dos tipos: i) facet_grid and ii) facet_wrap

Faceteado con facet_wrap

  geom_point() +

Faceteado con facet_grid

  geom_point() +

Cuando substitu√≠mos geom_point() por otro geom obtenemos un plot distinto. Los geom m√°s comunes:

Suavizar el scatterplot

Si el scatterplot tiene mucho ruido se puede usar un suavizado para facilitar la visualizaci√≥n del patr√≥n dominante usando geom_smooth().

Distintos m√©todos de suavizado por ejemplo: lm, glm, gam, loess y rlm

  geom_smooth()

Incluir un suavizado lineal

Incluir un suavizado lineal
