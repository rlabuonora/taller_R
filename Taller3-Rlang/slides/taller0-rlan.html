Environments

  • Son el contexto en el que se evalan las expresiones.
  • Hay una tabla de correspondencia entre el smbolo y el valor

Error

Si un smbolo no est definido en el current environment, R lo busca en su "padre".

num <- 1
f <- function() {
  num
}
f()
## [1] 1

num <- 1
f <- function() {
  num <- 2
  num
}
f()
## [1] 2
num
## [1] 1

Si el nombre no est definido en ningun environment.

  xyz +1
